## Project 1: How Does Hardware Work?

This was a **group project** where we explored the basics of digital logic.

### What we built:
- NOT Gate
- AND Gate
- OR Gate
- XOR Gate
- NAND Gate

By designing these gates, we learned how hardware logic forms the base of computing systems.
